# Draggable Component Configuration (Using [SortableJS](https://github.com/SortableJS/Sortable))

`Draggable` is a component used for reorder drag-and-drop lists. Its purpose is to drag from one list to another or within the same list using `md-draggable` and `md-draggable-item` components.

By default `md-draggable` component specifies slotable `md-draggable-item` components that can be inserted into and should be draggable. Also by default `Draggable` component disallow sorting inside list or between lists.
To change this, please set property `sort` to attributes in markup to `md-draggable`.

The `SortableJS` has many configuration options. Using different combinations of those properties. The `Draggable` component exposes these options via Lit-Element properties and attributes.
** Please Note that by default all `Boolean` properties set to `false` and all `String` properties set to `""` (empty string) **:

## ** group (by default `null`) ** ##

_Object property_. To drag elements from one list into another, both lists must have the same group value. You can also define whether lists can give away, give and keep a copy (clone), and receive elements.

- name: `String` — group name
- pull: `true|false|["foo", "bar"]|'clone'|function` — ability to move from the list. clone — copy the item, rather than move. Or an array of group names which the elements may be put in. Defaults to true.
- put: `true|false|["baz", "qux"]|function` — whether elements can be added from other lists, or an array of group names from which elements can be added.
- revertClone: `boolean` — revert cloned element to initial position after moving to a another list.

Example:

```
 <md-draggable sort .group=${{ name: "LeftList", pull: true, put: ["RightList] }}>
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item4</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```
or

 ```
 <md-draggable sort group="ListGroup">
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item4</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```

## ** sort ** ##

_Boolean property_. Allow sorting inside list.

Example:
 ```
 <md-draggable sort>
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item4</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```

## ** delay ** ##

_Number property_. Time in milliseconds to define when the sorting should start. Unfortunately, due to browser restrictions, delaying is not possible on IE or Edge with native drag & drop.

## ** animation (in ms time) ** ##

Number property. Animation speed moving items when sorting, `0` — without animation.

## ** handle ** ##

_String property_. To make list items draggable, Sortable disables text selection by the user. That's not always desirable. To allow text selection, define a drag handler, which is an area of every list element that allows it to be dragged around.

Example:

 ```
 <md-draggable sort handle="md-icon[name='icon-drag_16']">
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item4</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```

## ** filter ** ##

_String property_. Selectors that do not lead to dragging.

Example:

 ```
 <md-draggable sort filter="md-draggable-item[disabled]">
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item4</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```

## ** direction (will be `vertical` if not given) ** ##

_String property_. Direction of sortable items

## ** draggable (`md-draggable-item` by default) ** ##

_String property_. Specifies which items inside the element should be draggable.

Example:

```
 <md-draggable draggable-items="md-draggable-items">
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item4</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```

## ** forceFallback ** ##

_Boolean property_. Ignore the HTML5 DnD behaviour and force the fallback to kick in.

Example:

```
 <md-draggable force-fallback>
   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)
   </md-draggable-item>
   <md-draggable-item slot="draggable-item" disabled>Sortable Item2 (Disabled)</md-draggable-item>
   <md-draggable-item slot="draggable-item">Sortable Item3</md-draggable-item>

   <md-draggable-item slot="draggable-item">Sortable Item5</md-draggable-item>
 </md-draggable>
```

_String property_. Class name for the cloned DOM Element when using forceFallback.

Example:

   <md-draggable-item slot="draggable-item">
     <md-icon name="icon-drag_16"></md-icon>Sortable Item1 (Handle)

***

`Draggable` component contain event listeners that are fired when a specific option is updated. The table describe when event is calling (To see object detail, please follow link in `Event Type` table section):

| Event | Description | Event Type |
| :-: | :-: | :-: |
| drag-change | Dragging element changes position | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-clone | Creating a clone of element | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-unchoose | Element is unchosen | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-choose | Element is chosen | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-remove | Element is removed from the list into another list | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-add | Element is dropped into the list from another list | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-end | Element dragging ended | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-start | Element dragging started | [SortableEvent](https://github.com/SortableJS/Sortable#event-object-demo) |
| drag-move | Event when you move an item in the list or between lists | [MoveEvent](https://github.com/SortableJS/Sortable#move-event-object) |
